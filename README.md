# Barker AI Financial Intelligence

Volca-inspired financial intelligence app for Barker and Sons Plumbing.

## Demo accounts

- Admin: `admin@barker-ai.local` / `BarkerAdmin!2026`
- Executive: `brenda@barker-ai.local` / `BarkerExec!2026`
- Finance/Ops: `cat@barker-ai.local` / `BarkerOps!2026`

## Required environment

Copy `.env.example` to `.env.local` for local development.

Real AI responses require `OPENAI_API_KEY`. The app will not fake AI answers when the key is missing.

## Local run

```bash
npm install
npm run dev
```

## Product focus

The first version is centered on Barker's Fathom-derived ServiceTitan workflow:

- executive financial health
- closeout-to-payroll bottlenecks
- service-manager review backlog
- payment and open-balance risk
- AI analyst chat
- reports, alerts, integration placeholders, and admin settings
